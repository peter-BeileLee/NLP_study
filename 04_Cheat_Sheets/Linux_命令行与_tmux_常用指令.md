## 🖥️ Linux 常用命令行

### 1. 系统与进程管理

* **查看系统运行状态/进程**：`top` 或 `htop`
* **查看磁盘剩余空间**：`df -h`
* **查看内存使用情况**：`free -m`
* **结束指定进程**：`kill <PID>`（强制结束：`kill -9 <PID>`）

### 2. 网络与连接

* **测试网络连通性**：`ping <ip/domain>`
* **查看网络端口占用**：`netstat -tuln` 或 `ss -tuln`
* **远程连接服务器**：`ssh user@hostname`
* **下载网络文件**：`wget <URL>` 或 `curl -O <URL>`

### 3. 权限管理

* **以管理员权限执行**：`sudo <command>`
* **修改文件权限**：`chmod 755 <file>`
* **修改文件所属用户**：`chown user:group <file>`

## 🧵 tmux 常用指令

`tmux` 用于在终端中开启多窗口分屏，即使 SSH 断开，后台任务也不会停止。

### 1. 会话 (Session) 管理（在常规终端输入）

| 指令 | 功能描述 |
| --- | --- |
| `tmux new -s <name>` | 创建一个名为 `<name>` 的新会话 |
| `tmux ls` | 列出当前所有的 tmux 会话 |
| `tmux a -t <name>` | 接入/恢复名为 `<name>` 的会话 |
| `tmux kill-session -t <name>` | 关闭/销毁名为 `<name>` 的会话 |

### 2. 会话内操作快捷键（必须先按前缀键 `Ctrl + B`）

进入 tmux 会话后，先按下 `Ctrl + B` 松开，再按下方的对应按键：

#### 会话与窗口控制

* **分离当前会话**（后台运行）：`Ctrl + B`，然后按 `D`
* **新建窗口** (Window)：`Ctrl + B`，然后按 `C`
* **切换到下一个窗口**：`Ctrl + B`，然后按 `N`
* **关闭当前窗口**：`Ctrl + B`，然后按 `&`

#### 窗格 (Pane) 分屏操作

* **左右平分布局**（垂直分屏）：`Ctrl + B`，然后按 `%`
* **上下平分布局**（水平分屏）：`Ctrl + B`，然后按 `"`
* **切换到其他窗格**：`Ctrl + B`，然后按 `方向键(↑/↓/←/→)`
* **全屏/退出全屏当前窗格**：`Ctrl + B`，然后按 `Z`
* **关闭当前窗格**：`Ctrl + B`，然后按 `X`
