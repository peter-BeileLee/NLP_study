## 💻 正在进行的硬核实验：Java 高仿 SSH 服务器
为了深刻理解 NLP 大模型在分布式服务器上的部署原理，我正在用 Java 从零编写一个高仿版的 SSH/SCP 服务器：
- [ ] 用 Java `ServerSocket` 监听 TCP 端口。
- [ ] 在终端 PowerShell 中利用 `TcpClient` 远程向 Java 发送系统指令（模拟 SSH）。
- [ ] 通过特定前缀实现远程文件传输写盘（模拟 SCP）。
